# Crud-Project

## Update Environment variable.
```
NODE_ENV=development
DATABASE=test-task-db
DB_USERNAME=root
DB_PASSWORD=root
DB_HOST=localhost
DB_PORT=3306
```

## Start server

Install node module:

```bash
$ npm install
```
Create Database:
update dbconfig.js

```bash
$ npm run dbcreate
```

Database Migration:

```bash
$ npm run dbmigrate
```


Start Server:

```bash
$ npm start
```
